## What Each Workflow Does

### 🔨 Build Workflow
- Tests on Node.js 18.x AND 20.x (matrix)
- Validates ALL JSON files
- Verifies HTML structure exists
- Uploads artifacts for 7 days
- **When**: Every push to main/master + PRs
- **Time**: ~3-5 minutes

### 🚀 Deploy Workflow
- Builds your site
- Creates deployment package
- Deploys to GitHub Pages
- Validates post-deployment
- **When**: Every push to main/master
- **Manual**: Can trigger manually anytime
- **Time**: ~5-10 minutes total

### ✔️ Quality Workflow
- Validates ALL JSON files recursively
- Checks critical files exist
- Monitors file sizes
- Verifies script loading order
- **When**: Every push + PRs + daily 2 AM UTC
- **Time**: ~2-3 minutes
