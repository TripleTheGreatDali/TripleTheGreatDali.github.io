## 💡 IF SOMETHING GOES WRONG

### Issue: GitHub Actions shows error
**Solution**: 
1. Click the workflow in Actions tab
2. Scroll down to see the error
3. Most common: Wait 1-2 minutes and retry
4. Check network connection

### Issue: Site still loads slow
**Solution**:
1. Hard refresh: `Ctrl+Shift+R` (Windows) or `Cmd+Shift+R` (Mac)
2. Clear cache: `Ctrl+Shift+Delete` (Windows)
3. Try different browser
4. Wait 5-10 minutes for CDN to update

### Issue: Data not showing
**Solution**:
1. Check browser console (F12)
2. Check Network tab (are requests successful?)
3. Verify JSON files exist in `assets/data/`
4. Check file names match exactly

### Issue: Styling looks broken
**Solution**:
1. Hard refresh entire page
2. Clear browser cache completely
3. Try in private/incognito mode
4. Check mobile view vs desktop
